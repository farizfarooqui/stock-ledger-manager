# 📦 Stock Ledger Manager

A modern, fast, and responsive web application built to manage stock inventory and ledger entries efficiently. 

🚀 **Live Demo:** [https://stock-ledger-manager.vercel.app](https://stock-ledger-manager.vercel.app)

## ✨ Features
- **Real-time Tracking:** Easily track stock in and stock out.
- **Fast Performance:** Built with Vite for lightning-fast HMR and optimized builds.
- **Modern UI:** Clean and accessible user interface using Tailwind CSS and shadcn/ui.
- **Responsive Design:** Works perfectly on desktop, tablet, and mobile devices.

---

## 🛠️ Tech Stack
- **Frontend Framework:** [React](https://reactjs.org/) + [Vite](https://vitejs.dev/)
- **Language:** TypeScript / JavaScript
- **Styling:** [Tailwind CSS](https://tailwindcss.com/)
- **UI Components:** [shadcn/ui](https://ui.shadcn.com/)
- **Animations:** Framer Motion
- **Deployment:** [Vercel](https://vercel.com/)
